Sales Performance Analysis
Project Overview
This project analyzes sales data for a retail store to identify sales trends, top-performing products, and regional performance.
Tools Used
- Excel / Google Sheets
- SQL
- Tableau

Objectives
- Analyze monthly sales trends
- Identify top-selling products
- Compare sales performance by region

Data Cleaning
- Removed duplicates
- Handled missing values
- Created calculated fields for total sales
Key Insights
- Electronics category generated the highest revenue
- Sales peaked in November and December
- Northern region recorded the lowest sales
Recommendations
- Increase inventory for high-performing products
- Improve marketing strategies in low-performing regions
- Prepare inventory ahead of peak seasons
Deliverables
- SQL queries
- Excel analysis file
- Tableau dashboard
